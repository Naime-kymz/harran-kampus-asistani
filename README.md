# Harran Kampüs Asistanı

Harran Kampüs Asistanı, Harran Üniversitesi öğrencilerinin kampüs içerisindeki bilgilere tek bir yerden ulaşabilmesini sağlayan web tabanlı bir uygulamadır. Uygulama sayesinde duyurular, etkinlikler, ring saatleri, yemekhane menüsü ve kampüs haritası gibi bilgiler tek bir ekranda görüntülenebilir.

## Projenin Amacı

Üniversite kampüslerinde öğrenciler genellikle farklı platformlardan bilgi almak zorunda kalmaktadır. Bu proje, kampüs içi bilgileri tek bir uygulamada toplayarak öğrencilerin ihtiyaç duyduğu bilgilere daha hızlı ulaşmasını amaçlamaktadır.

## Özellikler

- Duyuru sistemi
- Etkinlik oluşturma ve listeleme
- Kampüs haritası üzerinde bina gösterimi
- Ring saatleri ve güzergah bilgileri
- Yemekhane menüsü görüntüleme
- Haftalık yemekhane değerlendirme sistemi
- Yönetici paneli üzerinden içerik ekleme

## Kullanılan Teknolojiler

- ASP.NET Core Web API
- C#
- Entity Framework Core
- MySQL
- HTML
- CSS
- JavaScript
- Leaflet (harita gösterimi)

## Projenin Çalıştırılması

Projeyi çalıştırmak için aşağıdaki adımlar izlenebilir:

1. Proje klasöründe terminal açılır.

2. Aşağıdaki komut çalıştırılır.

dotnet run

3. Uygulama çalıştıktan sonra tarayıcıdan aşağıdaki adres açılır.

http://localhost:5226

## Proje İçeriği

- Duyurular ve etkinlikler yönetici panelinden eklenebilir.
- Kampüs haritası üzerinde fakülteler ve önemli noktalar görüntülenebilir.
- Ring durakları ve saatleri listelenebilir.
- Günlük yemekhane menüsü görüntülenebilir.
- Öğrenciler yemekhane için değerlendirme yapabilir.
